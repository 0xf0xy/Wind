# About

**Wind** is a wordlist generator designed to exploit common password patterns, making it ideal for **dictionary attacks** and **realistic brute-force scenarios**.

<br>

# Installation

```bash
git clone https://github.com/0xf0xy/Wind.git
cd Wind
sudo pip install .
wind -h
```
